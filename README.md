## Overview
Setup Nvidia services

## Features
- Disable services
- Stop processes
- Remove $env:SystemDrive\Intel folder
- Remove diagnostics tracking scheduled tasks
- Disable Nvidia control panel
- Clean up old Nvidia driver folder
- Turn off Ansel
